Research Thesis Map

Current problem clusters, not publication nostalgia

My research orientation is architectural: identify which structure should remain explicit, which should be learned, and which should be delegated to constrained optimization loops. The objective is not maximal model complexity; it is maximal intelligibility per unit capability.

Cluster 1 — Cognitive Security

Security in contested environments is an epistemic problem before it is a classification problem. The central question is how to preserve calibrated judgment when ground truth, context, and mission profile are all nonstationary.

Cluster 2 — Graph Modeling & Representational Boundaries

Graph methods become substantially more useful when paired with explicit program structure, deterministic validators, and world-state interfaces. This is the core representational thesis behind my Augrade and assembly/twin framing.

Cluster 3 — Agent Epistemics and Control Planes

Agent harnesses should remain thin and legible while models absorb complexity. The practical metric is whether system complexity decreases as model capability increases.
